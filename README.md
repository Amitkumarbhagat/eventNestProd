# 📌 Project Overview

**EventNest** is a full-stack web application developed using the **MERN Stack** for managing and exploring events efficiently.  
The platform allows users to register, login securely, browse events, and manage event-related activities through a responsive and user-friendly interface.

This project was developed as part of the **CDAC Project Work** by:

- Amit Bhagat  
- Pragyaraj Solanki  
- Pratik Dhake  



# 🛠️ Tech Stack

## Frontend
- React.js
- React Router
- Axios

## Backend
- Node.js
- Express.js

## Database
- MongoDB

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/amitbhagatcmfeb26/EventNest.git
```

---

## 2️⃣ Move into Project Folder

```bash
cd EventNest
```

---

# 🔹 Backend Setup

## Install Dependencies

```bash
cd backend
npm install
```

---

## Create `.env` File

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
EMAIL_USER=your_email
EMAIL_PASS=your_password
```

---

## Start Backend Server

```bash
npm start
```

OR

```bash
nodemon server.js
```

---

# 🔹 Frontend Setup

## Open New Terminal

```bash
cd frontend
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Frontend

```bash
npm run dev
```

OR

```bash
npm start
```
